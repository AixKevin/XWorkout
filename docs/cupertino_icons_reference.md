# CupertinoIcons 官方文档参考

> 来源: https://api.flutter.dev/flutter/cupertino/CupertinoIcons-class.html

## 使用说明

CupertinoIcons 是 iOS 风格的图标库。要使用此图标库，请确保在 `pubspec.yaml` 中添加了 `cupertino_icons` 依赖：

```yaml
dependencies:
  cupertino_icons: ^1.0.0
```
